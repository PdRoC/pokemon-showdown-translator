# Pokémon Showdown Translator
Extensión de Chrome para traducir Pokémon Showdown al español.

## ¿Cómo contribuir?
1. Edita los archivos en la carpeta `data/` (CSV).
2. Ejecuta `python tools/compiler.py` para generar el nuevo diccionario.
3. Haz un Pull Request con tus cambios.

## Instrucciones de instalación
1. Descarga este repositorio.
2. En Chrome, ve a `chrome://extensions/`.
3. Activa "Modo de desarrollador" y haz clic en "Cargar descomprimida".
4. Selecciona la carpeta donde descargaste el proyecto.
